# TDD-Atividade
Sistema de Cálculo de Salário Líquido
Descrição

Sistema desenvolvido em Python para cálculo de salário líquido, aplicando corretamente os descontos de INSS e IRRF, seguindo os princípios da Engenharia de Software Moderna e o ciclo de Desenvolvimento Orientado a Testes (TDD) com apoio de Inteligência Artificial.

Tecnologias

Python
VS Code 


Atributos de Qualidade

Confiabilidade
Testabilidade
Manutenibilidade
Legibilidade

Métricas de Software

Cobertura de código
Complexidade ciclomática
WMC (Weighted Methods per Class)
CBO (Coupling Between Objects)
LOC (Lines of Code)

Testes (TDD)

Testes escritos antes da implementação
Cobertura de casos válidos, inválidos e limites
Garantia de evolução segura do código
