# TDD-Atividade  
Sistema de Cálculo de Salário Líquido.

Para execução dos testes, precisa executar o comando "python -m unittest discover", dentro da pasta "salario_liquido".

## Descrição
Este projeto consiste no desenvolvimento de um sistema em Python para o cálculo de salário líquido, realizando corretamente os descontos de INSS e IRRF. O sistema foi construído seguindo os princípios da Engenharia de Software Moderna, utilizando o ciclo de Desenvolvimento Orientado a Testes (TDD), com apoio de Inteligência Artificial durante o processo de desenvolvimento.

## Tecnologias Utilizadas
- Python  
- Visual Studio Code (VS Code)

## Atributos de Qualidade
- Confiabilidade  
- Testabilidade  
- Manutenibilidade  
- Legibilidade  

## Métricas de Software
- Cobertura de código  
- Complexidade ciclomática  
- WMC (Weighted Methods per Class)  
- CBO (Coupling Between Objects)  
- LOC (Lines of Code)  

## Testes (TDD)
- Testes definidos antes da implementação do código  
- Cobertura de cenários válidos, inválidos e casos de limite  
- Garantia de evolução segura e controlada do sistema
